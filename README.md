# 空运重量段成本预估器

这是一个纯前端空运泡比/重量段成本预估工具，支持单服务商计算、多服务商比价、分泡计费和附加费用成本拆分。

## 在线部署

项目可直接部署到 GitHub Pages。入口文件是 `index.html`，计算逻辑在浏览器端运行，不需要后端服务。

## 使用说明

- 输入实重、尺寸或总体积。
- 填写重量段价格和计费规则。
- 可启用分泡计费，支持五五分泡、四六分泡、三七分泡和自定义承担比例。
- 可启用附加费用，支持按 kg、按票、按 CBM、按主运费百分比、最低收费和手动总额。
- 点击计算按钮查看推荐重量段、结算计费重、主运费、附加费用和最终总成本。
- 多服务商比价会按每个服务商独立计费系数自动换算体积重、泡比和总成本。
- 起运地/目的地输入框支持机场三字码自动匹配。

## 数据说明

页面默认服务商和价格均为演示数据，不代表真实报价。
