# 项目改动日志

本文件用于记录空运泡比计算器的每次产品、代码、部署改动。后续每次修改项目时，都需要在提交前追加一条日志。

## 记录规则

- 每条日志包含：日期、提交、变更内容、影响文件、验证方式、部署状态。
- `estimator.html` 是主要开发文件，`index.html` 是 GitHub Pages 线上入口；涉及页面功能变更时，两者必须保持同步。
- 本地未提交的临时文档或需求文档不计入已发布改动，除非明确纳入提交。
